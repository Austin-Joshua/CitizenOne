# Citizen One — AI Unified Public Infrastructure Platform

Citizen One is an enterprise-grade, proactive citizen operating system designed to simplify bureaucracy, discover opportunities, and manage identity through a premium glassmorphism interface.

## 🚀 Vision
"One Intelligent Platform That Anticipates, Guides, and Empowers Every Citizen."

## 🛠️ Tech Stack
- **Frontend**: React, Vite, Tailwind CSS, Framer Motion, Lucide Icons.
- **Backend**: Node.js, Express, JWT, Helmet, Morgan.
- **Design**: Premium Glassmorphism UI (Custom CSS Design System).

## 📂 Project Structure
```
/citizen-one
├── frontend/             # React application
│   ├── src/
│   │   ├── components/   # UI Primitives & Layout
│   │   ├── pages/        # Dashboard, Login, AI Simplifier, etc.
│   │   ├── context/      # Auth & Theme Management
│   │   └── styles/       # Glassmorphism Design System
├── backend/              # Node.js API
│   ├── src/
│   │   ├── routes/       # Auth, AI, Users, Schemes
│   │   ├── middlewares/  # JWT & RBAC
│   │   └── app.js        # Server Entry
```

## 🔑 Key Features
1. **AI Bureaucracy Simplifier**: Guided procedural assistance.
2. **Dynamic Dashboard**: Personalized cards and proactive alerts.
3. **Identity Vault**: Secure document management.
4. **Benefit Discovery**: Tailored scheme matching.
5. **Admin Panel**: System-wide analytics and user management.

## 🚦 Getting Started

### Backend
1. `cd backend`
2. `npm install`
3. `npm start` (Runs on port 5000)

### Frontend
1. `cd frontend`
2. `npm install`
3. `npm run dev` (Runs on port 5173)
